Lambda Calculus Example

1 Overview

1.1 concept

  • Lambda expression: λ x.e
  • Application: (e e)

1.2 representation

  • ((λ(x) (λ y) (+ x y))) 1 2
  • (((λ x .λ y. (+ x y)) 1) 2)

1.3 β-reduction:

evaluate an expression by replacing the free variable.

((x (x)(y (y)(+x y))y )x 1)

  • λ x. λ y. (+ x y) 1
  • λ x. λ y. λ x (+ x y) 1 2 3

1.4 α-conversion

α-conversion: rename variable.

((x (x)(y (y)(+x y))y )x 1)

1.5 Order of evaluation

((λ x . (+ x 1)) (λ y . (+ y 1)) 2)

Can be:

  • (+ (λ y . (+ y 1) 2) 1)
  • (λ x . (+ x 1) (+ 2 1))

1.6 Examples

  1. Identity function

    (λ x.x) (λ y.y)

  2. Self application function

    λ x.(x x)

1.7 more complicated

  1. The expressoin

    (((λ f.λ x. (f x)) λ y.y) λ z.(z z))

    • g = λ y.y
    • v = λ z.(z z)
  2. Turn into
    • (λ f.λ x.(f x)) g v
    • (g v) = λ z.(z z)

1.8 Church encoding

  • numbers: 0, 1, 2, …
  • number fnction: succ, add
  • boolean: true, false
  • boolean function: ite, not

1.9 Numbers

zero
λ f.λ x.x
one
λ f.λ x.(f x)
two
λ f.λ x.(f (f x))
n

E.g. ((two g) z) == (g (g z))

1.10 Functions

succ
λ n. λ f. λ x. (f ((n f) x))
(no term)
(succ (succ zero))
add
λ m. λ n. λ f. λ x. ((((m succ) n) f) x)

1.11 boolean

true
λ x. λ y. x
false
λ x. λ y. y
ite
λ c .λ t. λ e. ((c t) e)

Examples:

(((ite true) s1) s2)
not a
(((ite a) false) true)
